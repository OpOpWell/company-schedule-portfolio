# 社内共有スケジュール管理システム

## なぜ作ったか

社内に情報共有の仕組みが存在しておらず、予定や電話対応、工事進捗などが口頭や紙のメモで管理されていました。

その結果、

* 電話折返し忘れ
* 出先予定の把握不足
* 工事進捗の共有不足

が発生していました。

そこで業務改善を目的として、Googleスプレッドシート・Google Apps Script（GAS）・AppSheetを活用し、社内共有システムを構築しました。

## システム概要

以下の情報を一元管理できるようにしました。

* 出先予定
* 工事予定
* 会議予定
* 作業状況
* 電話履歴
* 車検管理
* 一覧スケジュール

## 実装内容

### Google Sheets

各種情報を共有データベースとして管理。

### Google Apps Script

* 一覧スケジュール自動作成
* ダッシュボード集計
* 件数自動更新
* 未読件数集計
* 電話対応状況集計

を実装。

### AppSheet

スマートフォン・PCの両方から利用できるアプリとして構築。

* 外出先から入力可能
* 状況確認可能
* 既読管理可能
* 写真添付対応

## 工夫した点

* 現場業務でも利用しやすい入力項目設計
* 状態ごとのカラー表示
* ダッシュボードによる状況把握
* ノーコードとGASを組み合わせた運用設計

## 使用技術

* Google Sheets
* Google Apps Script (GAS)
* AppSheet

## 学習内容

独学でGoogle Workspace環境を学習し、業務改善を目的とした社内共有システムを設計・構築しました。
