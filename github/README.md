# 社内共有スケジュール管理システム

## 概要

Google Workspace（Google Sheets / Google Apps Script）を活用し、社内の予定・進捗・電話対応・車両管理を一元管理するための業務改善システムを開発しました。

複数の管理シートを一覧スケジュールへ自動集約し、要確認事項の抽出、既読管理、担当別集計、Googleカレンダー連携を実装しています。

業務で発生する情報共有漏れや確認漏れを防止し、低コストで運用できる社内向け管理システムを目指しました。

---

## 開発背景

社内では以下の情報が個別管理されていました。

* 出先予定
* 工事予定
* 会議予定
* 電話対応履歴
* 車検管理

その結果、

* 電話折返し忘れ
* 情報共有漏れ
* 車検期限の見落とし
* 担当者ごとの確認状況が不明

といった課題が発生していました。

そこで Google Workspace を利用し、情報を一元管理できるシステムを構築しました。

## 工夫した点

### 情報の一元化

複数シートの情報を一覧スケジュールへ自動集約し、全体状況を把握しやすくしました。

### 要確認事項の可視化

期限切れ・近日予定を自動抽出し、確認漏れを防止しています。

### Googleカレンダー連携

Googleカレンダーと連携することで、PC・スマートフォンの両方から予定確認が可能です。

### 社用車予約管理

予約重複を自動判定し、車両利用の競合を防止しています。

### アーカイブ機能

過去データを自動で整理し、月単位で閲覧できるようにしました。
